## Prerequisites
- The game must be bought via Steam
- [PortProton](https://flathub.org/en/apps/ru.linux_gaming.PortProton)
- [Frosty Mod Manager Linux Patch](https://www.nexusmods.com/masseffectandromeda/mods/1190)

## Step 1 - Initial start
- set the compatibility tool to **Proton Experimental**
- set the following launch options:
        `PROTON_DISABLE_NVAPI=1`
- in the EA installer, press <kbd>Customize installation</kbd> and disable <kbd>Create Desktop shortcut</kbd>
- wait until the game is started and quit it

## Step 2 - Preparing PortProton
- start PortProton
- go to <kbd>Wine Settings</kbd> → <kbd>Prefix Manager</kbd> → <kbd>Drives</kbd> → check <kbd>Show dot files</kbd> → <kbd>Ok</kbd>
- quit PortProton

## Step 3 - Preparing Frosty Mod Manager
- extract the zip file to a location that makes sense for you (e.g. `~/Apps`{: .filepath }/`~/Games`{: .filepath })
- run `FrostyModManager.exe`{: .filepath } with PortProton
- if you want, you can now create an Application/Desktop/Steam shortcut, by clicking on <kbd>Create Shortcut</kbd>
- press <kbd>Launch</kbd>

## Step 4 - Adding the game to Frosty Mod Manager
- In Frosty Mod Manager, press <kbd>Scan for games</kbd> and wait until it detects ME:C
  - If it doesn't find it automatically, you have to add the game manually by clicking on <kbd>Add</kbd> and then navigating to your ME:C installation directory (usually similar to `~/.local/share/steam/steam/steamapps/common/Mirrors Edge Catalyst`{: .filepath })
  - If the dot-folders don't show up, you have to create a symlink for the game directory

## Step 5 - Modding the game
- double click on the game in Frosty Mod Manager
- now you can drag and drop your downloaded mods (zip folders) into Frosty Mod Manager
- to apply them, click on every mod you want to apply and press <kbd>Apply mods</kbd>
- once all mods are applied, click on <kbd>Install mods</kbd>

## Step 6 - Starting the game
- Frosty Mod Manager will now show you which Launch Options you have to set in Steam
  - including our previous patch, this would be:
  ```
  WINEDLLOVERRIDES="winmm=n,b" PROTON_DISABLE_NVAPI=1 %command% -dataPath "ModData/Default"
  ```
- set these Launch Options and have fun!
